Python + SQLite

Gestor de Clientes

Lógica del Proyecto

Este sistema implementa una agenda completa con persistencia de datos mediante SQLite. Permite administrar una base de datos de clientes realizando operaciones CRUD (Crear, Leer, Actualizar y Eliminar) directamente desde la consola.

Utiliza un menú interactivo infinito que permite al usuario gestionar los registros de la tabla clientes de forma dinámica y segura.

Características Técnicas

  • Conexión a Base de Datos con sqlite3.
  • Consultas SQL parametrizadas para evitar inyecciones.
  • Manejo de flujo con while True.
  • Operaciones SQL: INSERT, SELECT, UPDATE, DELETE.
  • Persistencia de cambios mediante commit().
agenda_sqlite.py 

import sqlite3

# --- Conexión a la BBDD ---
conexion = sqlite3.connect("empresa.db")
cursor = conexion.cursor()

print("Programa agenda SQLite Dominique Farias")

# --- Bucle del Menú Principal ---
while True:
    print("Escoge una opción:\n1.-Crear cliente\n2.-Listar clientes\n3.-Actualizar clientes\n4.-Eliminar clientes\n5.-Salir del programa")
    
    opcion = int(input("Selecciona una opcion: "))
    
    # 1. Crear Cliente
    if opcion == 1:
        nombre = input("Introduce el nombre del cliente:")
        apellidos = input("Introduce los apellidos del cliente:")
        email = input("Introduce el email del cliente: ")
        
        cursor.execute("""
            INSERT INTO clientes (nombre, apellidos, email) VALUES (?, ?, ?);
        """, (nombre, apellidos, email))
        conexion.commit()
        
    # 2. Listar Clientes
    elif opcion == 2:
        cursor.execute('SELECT * FROM clientes;')
        filas = cursor.fetchall()
        for fila in filas:
            print(fila)

    # 3. Actualizar Cliente
    elif opcion == 3:
        identificador = input("Introduce el identificador a actualizar:")
        nombre = input("Introduce el nombre del nuevo cliente:")
        apellidos = input("Introduce los apellidos del nuevo cliente:")
        email = input("Introduce el email del nuevo cliente: ")
        
        cursor.execute("""
            UPDATE clientes SET
            nombre = ?,
            apellidos = ?,
            email = ?
            WHERE Identificador = ?;
        """, (nombre, apellidos, email, identificador))
        conexion.commit()
            
    # 4. Eliminar Cliente
    elif opcion == 4:
        identificador = input("Introduce el identificador a eliminar: ")
        cursor.execute("""
            DELETE FROM clientes WHERE Identificador = ?;
        """, (identificador,))
        conexion.commit()

    # 5. Salir
    elif opcion == 5:
        print("byebye")
        exit()
