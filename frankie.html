Visor de Clientes

Lógica del Proyecto

Esta aplicación de escritorio utiliza la librería gráfica Tkinter para crear una interfaz de usuario visual. El programa establece una conexión directa con una base de datos MySQL local.

Mediante consultas SQL, extrae la información de la tabla "clientes" y la renderiza dinámicamente en un widget de tipo Treeview (tabla), permitiendo visualizar DNI, nombre, apellidos y email de forma organizada.

Características Técnicas

  • Interfaz gráfica (GUI) con tkinter.
  • Conexión a Base de Datos con mysql.connector.
  • Visualización de datos mediante ttk.Treeview.
  • Ejecución de consultas SQL (SELECT).
  • Iteración de resultados con bucles for.
clientes_gui.py 

import tkinter as tk
from tkinter import ttk
import mysql.connector

# --- Conexión a Base de Datos ---
conexion = mysql.connector.connect(
    host="localhost",
    user="empresadam",
    password="Empresadam123$",
    database="empresadam"
)
cursor = conexion.cursor()

# --- Configuración de la Ventana ---
ventana = tk.Tk()
arbol = ttk.Treeview(ventana, columns=("dninie","nombre", "apellidos","email"), show="headings")

# --- Definición de Cabeceras ---
arbol.heading("dninie", text="DNI del cliente")
arbol.heading("nombre", text="Nombre del cliente")
arbol.heading("apellidos", text="Apellidos del cliente")
arbol.heading("email", text="Email del cliente")

# --- Consulta SQL y Llenado de Tabla ---
cursor.execute('''SELECT * FROM clientes;''')
filas = cursor.fetchall()

for fila in filas:
    # Insertamos los datos recuperados en el árbol
    arbol.insert("", "end", values=(fila[1], fila[2], fila[3], fila[4]))

arbol.pack(padx=20,pady=20)

# --- Bucle Principal ---
ventana.mainloop()
